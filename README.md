## Description

##### A Web-hosted version of the NPUNGLR8R CLI tool for generating form emails.

### Installation

##### No install necessary! Hosted on GitHub pages, even functional on mobile!

### Usage

##### Select NPU, meeting date and the forms you need - NPUNGL0Tron generates one-click-copyable, preformatted emails for pasting, or for emails with preset recipients, click on the heading to open a prefilled email in your default email client!
